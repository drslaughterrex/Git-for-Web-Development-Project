Stretch Questions

    1. What is the difference between an inline element and a block element?
    A block element will always start on a new line, while an inline element does not.

    2. What happens when an element is positioned absolutely? 
    The element is no longer part of the typical layout flow of the document and is separated to it's own layer.

    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width?
    With the flex property.

    A) <div>, <h1>, <p>
    B) <a>, <img>
    c) <span>

    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
    
    The box model is the visual representation of how the content of the page can be structured relative to each element. consisting of (from outermost to innermost sections) Margin> Border > Padding > Content, you can use properties to customize each aspect as desired.
